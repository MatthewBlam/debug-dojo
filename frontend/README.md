# Frontend

Debug Dojo's frontend is a Next.js App Router application with
TypeScript, Tailwind CSS, and shadcn/ui.

## Local Development

Use the Node.js version from the root `.nvmrc`.

```sh
cd frontend
pnpm install
pnpm dev
```

The local app runs at <http://localhost:3000>.

## Checks

```sh
pnpm lint
pnpm typecheck
```
